# fx-future-forecast
必勝！FX未来予想チャート徹底比較についてのリポジトリです

# Develop

## 初期設定・インストール
```.sh
git clone git@github.com:DreamHackers/fx-future-forecast.git
cd fx-future-forecast
./setup.sh
```

## 起動
```.sh
node app.js
```
[こちら](http://localhost:9000/)にアクセスしてください
